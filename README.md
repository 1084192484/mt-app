##项目涉及到技术栈：
- vue全家桶：Vue、Vue-router、Vue-cli等
- 组件化：单Vue文件
- 模块化：ES6 Module
- 第三方模块：better-scroll axios等
- 基础开发环境和包管理：Node、npm
- 构建工具：webpack
- 编辑器：webstrom





## Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8081
npm run dev

# build for production with minification
npm run build

```
